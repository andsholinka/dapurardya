# Changelog - AI Scanner Feature

## [1.0.0] - 2026-03-17

### 🎉 Initial Release

#### Added
- AI-Powered Image Recognition menggunakan Gemini Vision
- Smart Recipe Matching dengan nutrition info
- Interactive Chat dengan Chef AI
- Credit system (2 credits per scan)
- Camera support untuk mobile & desktop
- Upload foto dari galeri
- Image compression dan validation
- Brightness detection
- Responsive design untuk semua devices
- Loading states dan animations
- Error handling yang comprehensive
- Scanner tips dan guide

#### Features
- **Scan Ingredients**: Deteksi bahan dari foto dengan confidence score
- **Recipe Suggestions**: Matching dengan database resep + match score
- **Nutrition Info**: Estimasi kalori dan makronutrien (P/C/F)
- **Chef AI Chat**: Panduan memasak step-by-step
- **Credit System**: 2 credit per scan, chat gratis
- **Premium Benefits**: Unlimited untuk admin

#### Technical
- Frontend: Next.js 16, TypeScript, Tailwind CSS
- Backend: Next.js API Routes, MongoDB
- AI: Gemini 2.0 Flash Exp (Vision) + 2.5 Flash (Text)
- Image Processing: Canvas API, compression, validation
- Authentication: JWT-based member system
